GitBook

GitBookとは

markdownで書いたドキュメントをhtmlやpdfなどで読みやすい形式に変換をかける機能を提供してくれるコマンドラインツールです。

インストール

  1. node.jsインストール

    GitBookインストールにはnpmを使用する必要がある為、node.jsをインストール。

  2. GitBookインストール

    以下コマンドを実行する

     npm install -g gitbook-cli

GitBookコマンド

  1. 雛形ファイル生成
    雛形ファイルが生成される。 
    gitbook init
    　
  2. ビルド
    MarkDownで記述したファイルがビルドされる。 
    gitbook build
    　

  3. ローカルサーバー起動
    ローカルサーバーが立ち上がります。

    gitbook serve


    http://localhost:4000にて確認可能。

  4. PDF出力
    PDFファイルが生成される。
    ※calibreをインストールする必要があります。

    gitbook pdf
